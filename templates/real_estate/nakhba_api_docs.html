المصادقة (Authentication)

كل طلب يتم إرساله للواجهة يجب أن يحتوي على إحدى الترويستين (Headers) التاليتين للتحقق من الصلاحية:

1 مفتاح API (API Key)

الموصى به للتطبيقات والأنظمة الخارجية.

X-API-Key: nkb_live_xxxxxxxxxxxxxxxxxx

2 JWT Token

لمستخدمي الموقع المسجلين (داخلي).

Authorization: Bearer <supabase_jwt>

قائمة المباني

يرجع قائمة بكافة المباني مع إحصائيات الإشغال والوحدات والإيراد المتوقع لكل مبنى.

مثال الاستجابة (JSON)

{
  "data": [
    {
      "number": 1,
      "type": "محلات قطع غيار السيارات",
      "total_units": 24,
      "rented_units": 12,
      "available_units": 9,
      "expected_annual_revenue": 1440000
    }
  ],
  "count": 13
}
GET /units
Scope: read
